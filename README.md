# Mjupa Chat - App ya Kuchati ya Siri

Mjupa Chat ni app ya kuchati ya siri inayofanya kazi duniani kote.

## Vipengele
- Kuchati na watu kwa siri
- Kutuma picha na video
- PIN ya usalama
- Bure kabisa

## Pakua App
https://sadat-ctrl.github.io/Mjupa-chat

## Jinsi ya kutumia
1. Fungua link hapo juu
2. Ongeza kwenye Home Screen
3. Ingia na PIN yako
4. Anza kuchati!
